# QR Scanner Testing Project

## Project Overview
This project contains testing documentation for the "Barcode and QR Scanner" mobile application.

The goal was to perform manual testing, identify defects, and document them following QA best practices.

---

## Test Environment
- Device: Google Pixel 8
- OS: Android 16
- Testing type: Manual testing

---

## Scope of Testing
The following features were tested:

- QR code scanning
- Barcode scanning
- QR code generation
- History functionality
- UI/UX behavior

---

## Bugs Summary
A total of 14 bugs were identified during testing:

- Critical: 2
- High: 2
- Medium: 5
- Low: 2
- Improvement: 3
